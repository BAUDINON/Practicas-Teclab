# Mis Prácticas de Teclab

Estas son mis prácticas de la carrera. Desarrollé una página web integral que abarca desde la interfaz de usuario hasta la persistencia de los datos.

## Tecnologías utilizadas

* **Lenguaje:** JavaScript | HTML | CSS
* **Frontend:** React.js
* **Backend:** Express.js | Sequalize
* **Base de datos:** PostgreSQL

Infraestructura (Docker)
El proyecto está íntegramente montado en contenedores de Docker. Esto significa que tanto el servidor como la base de datos están configurados para correr de forma aislada, asegurando que el proyecto funcione en cualquier computadora sin necesidad de instalar dependencias manualmente.
